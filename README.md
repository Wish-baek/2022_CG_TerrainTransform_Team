
# cg_terrain_project_team9

저희 조는 p5.js web editor 로 WEBGL로 프로젝트를 진행하였으며 
먼저 지난번 terrain 예제 소스코드를 이용하여 3차원 지형을 생성하였습니다.

3차원 지형의 변형에서 주제는 밤바다와 바다를 비추는 등대로 구성하였습니다.
